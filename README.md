# 49th Parallel Racing Website

This is a complete static website starter for 49th Parallel Racing.

## Files
- index.html — homepage
- updates.html — build updates page
- sponsors.html — sponsor/funding page
- contact.html — contact/sponsor interest page
- styles.css — shared site styling

## How to preview
Open index.html in a browser.

## How to edit
Use any text editor:
- Change team member names in index.html
- Change funding progress amounts in index.html and sponsors.html
- Replace placeholder emails/social handles
- Add sponsors to sponsors.html
- Add new updates to updates.html

## How to host
Best options:
- GitHub Pages
- Netlify
- Cloudflare Pages

Google Sites cannot fully import this as native editable pages. You can use it as a visual guide or embed individual pages, but coded hosting will preserve the design best.
